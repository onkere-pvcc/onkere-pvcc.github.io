Programming Terms

Term Definition
Debugging The process of identifying and resolving errors or bugs in a computer program or code.
Assembler A program that translates assembly language code into machine language code.
Assembly language A low-level programming language that is a symbolic representation of machine language code.
Translator A general term for a program that converts one form of code or data into another, including assemblers, compilers, and interpreters.
Compiler A program that translates high-level programming language code into machine language code, usually resulting in an executable file.
Interpreter A program that directly executes high-level programming language code without first converting it into machine language code.
Cross-platform software Software that can run on multiple operating systems or platforms without requiring modification.
Natural language Any human language, such as English or Chinese.
Formal language A language with a precisely defined set of rules and syntax, often used in mathematics and computer science.
Machine independence The ability of a program or code to run on different types of hardware and operating systems without modification.
High-level programming language A programming language designed to be easier to read and write than low-level languages, such as assembly language or machine language.
Object-oriented programming (OOP) A programming paradigm that emphasizes the use of objects, classes, and inheritance to organize and structure code.
Scripting language A type of high-level programming language that is often used for automation tasks or as a glue language to connect different programs or systems.
Integer A type of data that represents whole numbers, such as -3, 0, or 42.
Float A type of data that represents decimal numbers, such as 3.14 or -0.5.
String a type of data that represents text or characters, such as "hello" or "world".
17. Primitive data type a basic type of data that is built into a programming language, such as integers, floats, and strings.
Data structure a way of organizing and storing data in a program, such as lists, sets, or dictionaries.
Python collection data structures built-in data structures in the Python programming language that allow for organizing and manipulating collections of data, including lists, tuples, sets, and dictionaries.
List a collection data structure in Python that represents an ordered sequence of values.
Tuple a collection data structure in Python that represents an ordered, immutable sequence of values.
Set a collection data structure in Python that represents an unordered collection of unique values
Dictionary a collection data structure in Python that represents a mapping between keys and values.
Function a reusable block of code in a program that performs a specific task or set of tasks.
Local variable a variable that is defined and used within a specific function or block of code.
Global variable a variable that is defined and can be accessed from anywhere in a program.
Source program the original code written by a programmer in a high-level programming language.
Object program the compiled or interpreted code that is executed by a computer, often in the form of an executable file or binary code.
